# Repository agent skills

Canonical, vendored agent skills for this project live here:

```text
.agents/skills/
```

These folders are the **source of truth**. Harness-specific copies under `.cursor/skills/`, `.codex/skills/`, and `.claude/skills/` are generated by sync and must not be edited by hand.

See `docs/design/SKILL_SOURCES.md` for commit SHAs, licenses, and update instructions.

## Commands

```bash
npm run skills:sync   # copy approved skills into harness directories
npm run skills:check  # validate canonical + harness copies
```

Do not edit generated harness copies. Change `.agents/skills/<name>/`, then sync.
